# 📌 Pure Organic – Checklist

## 🧾 Qeydiyyat
- Yeni user qeydiyyatdan keçir  
- Eyni email ilə yenidən qeydiyyat olmur  
- Password qaydalara uyğundur (uzunluq və s.)  
- Password və confirm password eynidir  
- Boş xanalar error verir  
- Email düzgün formatda olmalıdır  

---

## 🔐 Login
- Düzgün məlumatla daxil olmaq olur  
- Səhv məlumatda error çıxır  
- “Show password” işləyir  
- “Forgot password” işləyir  
- Şifrə gizli görünür  
- Logout işləyir  

---

## 🔍 Axtarış
- Yazılan sözə uyğun nəticə çıxır  
- Nəticə yoxdursa mesaj çıxır  
- Böyük/kiçik hərf fərq etmir  
- Yarım sözlə axtarış işləyir  
- Axtarış nəticələri tez gəlir  
- Enter və search button işləyir  

---

## 📂 Kateqoriyalar
- Kateqoriyalar siyahısı görünür  
- Kateqoriyaya klik edəndə açılır  
- Uyğun məhsullar göstərilir  
- Məhsullar aydın göstərilir  

---

## 🎛 Filter / Sort
- Qiymətə görə filter işləyir  
- Birdən çox filter işləyir  
- Filter sıfırlamaq olur  
- Sort funksiyası işləyir  

---

## 📦 Məhsul detalları
- Məhsul adı və qiyməti düzgün göstərilir  
- Şəkillər dəyişmək olur  
- Məhsul təsviri var  
- Stok vəziyyəti göstərilir  
- Add to cart işləyir  
- Say seçmək olur  

---

## 🛒 Səbət (Cart)
- Məhsul səbətə əlavə olunur  
- Eyni məhsul əlavə ediləndə say artır  
- Məhsullar düzgün göstərilir  
- Say artırıb azaltmaq olur  
- Məhsulu silmək olur  
- Total price düzgün hesablanır  
- Səbət boşdursa mesaj çıxır  

---

## 💳 Checkout
- Çatdırılma məlumatları daxil edilir  
- Ünvan əlavə etmək olur  
- Sifariş xülasəsi göstərilir  
- Çatdırılma üsulu seçilir  
- Promo code tətbiq olunur  

---

## 💰 Ödəniş (Payment)
- Kart məlumatı daxil edilir  
- Yanlış kart error verir  
- Düzgün kartla ödəniş edilir  
- Uğurlu ödəniş mesajı çıxır  
- Sistem sifariş yaradır  

---

## ⚠️ Edge Cases
- Internet kəsildikdə error mesajı göstərilir  
- Boş data daxil edildikdə error çıxır  

---

## 📌 Nəticə
Checklist sistemin əsas funksiyalarını yoxlamaq üçün istifadə olunur və test prosesini sürətləndirir.
