## 📖 Layihənin Təsviri
Bu layihə organic qida məhsulları satan e-commerce veb saytının test edilməsinə yönəlib.  
Əsas məqsəd katalog modulunun (məhsul siyahısı, filter və sorting funksiyaları) düzgün işlədiyini və istifadəçi tələblərinə cavab verdiyini yoxlamaqdır.

---

## 🎯 Layihənin Əhatə Dairəsi (Scope)
Bu layihəyə aşağıdakılar daxildir:

- Məhsul katalogu funksionallığı  
- Kateqoriya üzrə filterləmə  
- Məhsulların sıralanması (qiymət, ad və s.)  
- Katalog səhifəsinin UI davranışı  

---

## 👥 İstifadəçi Rolları
- Qeydiyyatsız istifadəçi (Guest)  
- Qeydiyyatdan keçmiş istifadəçi (Registered User)  

---

## ⚙️ Test Edilən Funksiyalar
- Məhsul siyahısına baxış  
- Məhsulları kateqoriyaya görə filterləmək  
- Məhsulları sıralamaq  
- UI-nin responsivliyini yoxlamaq  

---

## 🧪 Test Növləri
- Funksional test  
- UI test  
- Exploratory test  

---

## 🌐 Test Mühiti
- Brauzer: Google Chrome  
- Cihaz: Desktop və Mobil  
- Əməliyyat sistemi: Windows  

---

## 🛠 İstifadə olunan alətlər
- Qase (Test Management Tool)  
- GitHub (Versiya nəzarəti)  

---

## 📂 Layihə Sənədləri
Bu layihədə aşağıdakı BA və QA sənədləri mövcuddur:

- User Story  
- Acceptance Criteria  
- Test Plan  
- Test Case-lər  
- Checklist  
- Test Summary Report  

---

## 🚀 Necə yoxlamaq olar
1. Saytı aç  
2. Katalog səhifəsinə keç  
3. Filter və sorting funksiyalarını istifadə et  
4. Nəticələri yoxla  

---

## 🐞 Bug Reporting
Test prosesi zamanı aşkar olunan bug-lar sənədləşdirilmişdir.
